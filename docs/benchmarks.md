- **§7.1 Binoculars drop** — install the optional extra and the upstream package:
    ```bash
    pip install humanize-en[perplexity]
    pip install git+https://github.com/ahans30/Binoculars
    ```
    Requires a CUDA / MPS device and ~14 GB of Falcon-7B weights on first use.

    pip install humanize-en[bench]

- **All gates** — set one of `OPENAI_API_KEY` / `ANTHROPIC_API_KEY` / `DEEPSEEK_API_KEY` (etc.) to exercise the LLM polish path.
